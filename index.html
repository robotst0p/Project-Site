Hackathon Protest-Finder!

2020 Ohio State Hackathon project. Built in two days, allows users to enter protests, view a live map of other entered protests and users can click on map markers to pull info relating to the entered protest. Backend was handeled using PHP and a MySQL database.

  • HTML
  • CSS
  • Javascript
  • PHP
  • MySQL
  • Google Maps Javascript API
  • Geocoder API
